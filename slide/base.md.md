## Marpとは

- Markdown でプレゼンテーションスライドを作成することができるツール
- 組み込みテーマや CSS テーマによるデザインを適用して、HTML, PDF, PowerPoint などの各種形式でエクスポートすることができる

---

## 書き方

- スライドの区切りは `---` 。

```Markdown
---
marp: true
---

# タイトル

---

## スライド1枚目

    ・
    ・
    ・
```
